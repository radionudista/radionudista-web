### Conceptos Clave

- **Contenido Multilingüe y Dinámico**: El contenido se gestiona como Markdown por idioma, se indexa automáticamente y está disponible en navegación y rutas. Agrega nuevos idiomas/contenido fácilmente.
- **Plugins Vite Personalizados**: El build multilenguaje y el indexado de contenido se manejan automáticamente en tiempo de build.
- **Debug y Extensibilidad**: Barra de debug y contextos para desarrollo y troubleshooting. El sistema está diseñado para fácil extensión.
